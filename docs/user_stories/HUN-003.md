# HUN-003 - Gestionar Perfil

**Descripción:**  
Como cliente del sistema, quiero poder actualizar mi información personal como nombre, correo y contraseña, para mantener mis datos de acceso correctos y actualizados dentro de NocheLista en todo momento.

### Criterios de Aceptación
- [ ] El cliente accede a su perfil desde cualquier pantalla mediante el ícono de usuario en el header  
- [ ] El cliente actualiza nombre y correo con validación de formato en tiempo real  
- [ ] El cliente cambia contraseña ingresando la actual y la nueva (mínimo 8 caracteres)  
- [ ] Si la contraseña actual es incorrecta se muestra error sin actualizar nada  
- [ ] Los cambios se guardan en Supabase y se reflejan inmediatamente en la interfaz  
- [ ] Solo el propio usuario puede editar su perfil (RLS activo)  
- [ ] Se muestra mensaje de confirmación tras cada actualización exitosa  

### Definición de Hecho (DoD)
- [ ] Código revisado por pares mediante pull request aprobado  
- [ ] Pruebas unitarias con cobertura ≥ 80% de la lógica de actualización  
- [ ] Política RLS usuario_editar_propio activa y verificada  
- [ ] Claves API en variables de entorno  
- [ ] Sin bugs de prioridad alta o media abiertos  
- [ ] Interfaz responsiva verificada en web y móvil
